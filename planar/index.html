Display Over Other Apps

This permission allows the floating countdown timer to appear on top of every other app on your phone. When converted to a native app, it will show as a persistent overlay that cannot be dismissed.

For Native App (Android):

1. Convert this HTML app using Cordova or Capacitor

2. The config.xml file included requests SYSTEM_ALERT_WINDOW

3. Users grant permission in: Settings > Apps > PlanAlarm > Display over other apps

For Web / PWA (Now):

The floating timer works inside the app. For true system-level overlay, you must convert to a native Android app.
