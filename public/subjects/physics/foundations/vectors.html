1 Scalars vs Vectors

Physical quantities can be classified as either scalars or vectors based on whether they have direction.

Scalar Quantities

Have only magnitude (size/amount). No direction needed.

Examples:

  • • Mass (5 kg)
  • • Temperature (25°C)
  • • Time (10 seconds)
  • • Speed (60 km/h)
  • • Distance (100 m)
  • • Energy (500 J)

Vector Quantities

Have both magnitude AND direction.

Examples:

  • • Force (10 N east)
  • • Velocity (60 km/h north)
  • • Displacement (100 m left)
  • • Acceleration (5 m/s² upward)
  • • Momentum (20 kg·m/s forward)
  • • Weight (50 N downward)

Key Distinction

Distance vs Displacement: Distance is how far you travel (scalar). Displacement is your change in position from start to finish, including direction (vector). You might walk 100m but end up only 50m from where you started!

2 Representing Vectors

Vectors are represented graphically using arrows (also called directed line segments). The arrow conveys two pieces of information:

Length of Arrow

Represents the magnitude (drawn to scale)

Direction of Arrow

Shows the direction of the vector quantity

Vector Notation

Vectors are written in bold or with an arrow above:

  • • F or F⃗ represents a force vector
  • • v or v⃗ represents a velocity vector
  • • |F| or F represents the magnitude only (scalar)

3 Adding Vectors

When adding vectors, both magnitude and direction must be considered. There are two main methods:

Triangle (Head-to-Tail) Method

  1. Draw the first vector to scale
  2. Draw the second vector starting from the head of the first
  3. The resultant is drawn from the tail of the first to the head of the last

Parallelogram Method

  1. Draw both vectors from the same starting point (tail-to-tail)
  2. Complete the parallelogram by drawing parallel lines
  3. The diagonal from the common point is the resultant

Practice Questions

1. Classify each quantity as scalar or vector: (a) Speed, (b) Velocity, (c) Mass, (d) Force, (e) Temperature

Show Answer

Scalar: Speed, Mass, Temperature (have magnitude only)

Vector: Velocity, Force (have magnitude AND direction)

2. A person walks 3 km east, then 4 km north. What is their total displacement?

Show Answer

Using Pythagoras theorem: R² = 3² + 4² = 9 + 16 = 25

R = √25 = 5 km

Direction: tan⁻¹(4/3) = 53.1° north of east

3. What is the difference between distance and displacement?

Show Answer

Distance is a scalar - the total length of path traveled, regardless of direction.

Displacement is a vector - the straight-line distance from start to finish, with direction.

Example: Walking around a 400m track once = 400m distance, but 0m displacement (you end where you started).

4. Two forces act on an object: 30 N to the right and 20 N to the left. What is the resultant force?

Show Answer

Since they're in opposite directions, subtract:

R = 30 N - 20 N = 10 N to the right

When vectors are along the same line, add if same direction, subtract if opposite.

Key Takeaways

  • • Scalars have magnitude only; vectors have magnitude AND direction
  • • Vectors are represented by arrows: length = magnitude, direction = vector direction
  • • Add vectors using head-to-tail (triangle) or parallelogram methods
  • • The resultant is the single vector that has the same effect as the combined vectors
  • • Distance is scalar; displacement is vector. Speed is scalar; velocity is vector.
