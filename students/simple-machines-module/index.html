1) Levers — Concepts

A lever rotates about a fulcrum. Torque = Force × Distance (from fulcrum). The lever balances when torques on both sides are equal.

Left torque: 0 N·m

Right torque: 0 N·m

Net torque: 0 N·m

Move the fulcrum and change weights. Predict which side will tip.

Step-by-step

  1. Set the fulcrum in the center.
  2. Place equal weights at equal distances — the beam should balance.
  3. Move one weight further away or increase its force to create imbalance.
  4. Observe torques displayed and predict motion direction.

2) Inclined Plane — Concepts

On a frictionless incline, the component of gravity along the plane is g·sin(theta). Acceleration = g·sin(theta).

Downhill force: 0 N

Acceleration (no friction): 0 m/s²

Try low and high angles and watch how acceleration changes.

Step-by-step

  1. Set the angle to a small value — the block will accelerate slowly.
  2. Increase the angle — acceleration increases roughly with sin(theta).
  3. Change the mass — note that acceleration does not depend on mass for frictionless incline.
