Simple Machines — Interactive Learning Module

Files:
- index.html — the main tutorial with simulations and quiz
- style.css — styles
- script.js — client-side simulation logic

How to open
1. Open `index.html` in a modern browser (double-click or open with your browser).
2. Interact with the lever and inclined plane panels. Adjust sliders and try the quiz.

Notes and next steps
- The visuals are lightweight SVG so you can extend them easily.
- Possible additions: friction toggle, energy readout, more machine types (pulley, wheel & axle), additional quiz questions.
