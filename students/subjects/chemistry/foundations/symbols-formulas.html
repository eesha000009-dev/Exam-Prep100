Symbols & Formula Writing

Understanding chemical symbols, valency, and writing formulas for ionic and covalent compounds.

Learning Objectives

  • • State chemical symbols of common elements
  • • Define valency and know valencies of common elements
  • • Write correct formulas for simple ionic compounds
  • • Write formulas for covalent molecules
  • • Use brackets in formula writing correctly

1 Chemical Symbols

A chemical symbol is a shorthand way of representing an element. Most symbols consist of one or two letters from the element's name.

Rules for Writing Symbols:

  • • First letter is always CAPITAL
  • • Second letter (if any) is always lowercase
  • • Examples: Carbon, Hydrogen, Natrium (Sodium), Ferrum (Iron)

Latin Names

Some symbols come from Latin names: Na (Natrium = Sodium), K (Kalium = Potassium), Fe (Ferrum = Iron), Cu (Cuprum = Copper), Ag (Argentum = Silver), Au (Aurum = Gold), Pb (Plumbum = Lead).

2 Valency

Definition

Valency is the combining power of an element - the number of electrons it can lose, gain, or share to form compounds.

Element Symbol Valency Group
Hydrogen H 1 1
Sodium, Potassium Na, K 1 1
Magnesium, Calcium Mg, Ca 2 2
Aluminum Al 3 3
Carbon, Silicon C, Si 4 4
Nitrogen, Phosphorus N, P 3 5
Oxygen, Sulfur O, S 2 6
Chlorine, Fluorine Cl, F 1 7

Variable Valency

Some elements have more than one valency:

  • • Iron (Fe): Fe²⁺ (ferrous) or Fe³⁺ (ferric)
  • • Copper (Cu): Cu⁺ (cuprous) or Cu²⁺ (cupric)
  • • Lead (Pb): Pb²⁺ or Pb⁴⁺

3 Writing Formulas for Ionic Compounds

Ionic compounds form when metals transfer electrons to non-metals. The formula shows the simplest ratio of ions.

Steps for Writing Ionic Formulas:

  1. 1 Write the symbols of the ions (metal first)
  2. 2 Write their valencies (charges)
  3. 3 Cross over the valencies (swap them)
  4. 4 Simplify if needed

The "Swap and Drop" Method:

Na⁺

valency 1

+

Cl⁻

valency 1

→

NaCl

1:1 ratio

1. Magnesium + Chlorine

Mg²⁺ + Cl⁻ → Cross: Mg₁Cl₂ → MgCl₂

2. Calcium + Oxygen

Ca²⁺ + O²⁻ → Cross: Ca₂O₂ → Simplify → CaO

3. Aluminum + Oxygen

Al³⁺ + O²⁻ → Cross: Al₂O₃ → Al₂O₃

4. Sodium + Sulfate

Na⁺ + SO₄²⁻ → Cross: Na₂(SO₄)₁ → Na₂SO₄

4 Polyatomic Ions (Radicals)

Polyatomic ions are groups of atoms that carry a charge and act as a single unit. Use brackets when more than one is needed.

Name Formula Charge
Ammonium NH₄⁺ +1
Hydroxide OH⁻ -1
Nitrate NO₃⁻ -1
Carbonate CO₃²⁻ -2
Sulfate SO₄²⁻ -2
Phosphate PO₄³⁻ -3

Important: Using Brackets

When more than one polyatomic ion is needed, enclose it in brackets:

  • • Ca²⁺ + (OH)⁻ → Ca(OH)₂ ✓
  • • Not CaOH₂ ✗ (wrong!)
  • • Al³⁺ + (SO₄)²⁻ → Al₂(SO₄)₃

5 Formulas for Covalent Compounds

Covalent compounds form when non-metals share electrons. The formula shows the actual number of atoms in each molecule.

Common Covalent Molecules

  • • Water: H₂O
  • • Carbon dioxide: CO₂
  • • Methane: CH₄
  • • Ammonia: NH₃
  • • Hydrogen chloride: HCl

Diatomic Elements

  • • Hydrogen: H₂
  • • Nitrogen: N₂
  • • Oxygen: O₂
  • • Chlorine: Cl₂
  • Also: F₂, Br₂, I₂

Remember

For covalent compounds, use prefixes to indicate numbers: mono (1), di (2), tri (3), tetra (4), penta (5). Example: Carbon dioxide (CO₂), Dinitrogen trioxide (N₂O₃).

Practice Problems

1. Write the formula for: (a) Sodium oxide, (b) Magnesium chloride, (c) Aluminum chloride

Show Solution

(a) Na⁺ + O²⁻ → Na₂O

(b) Mg²⁺ + Cl⁻ → MgCl₂

(c) Al³⁺ + Cl⁻ → AlCl₃

2. Write formulas for: (a) Calcium hydroxide, (b) Aluminum sulfate, (c) Ammonium nitrate

Show Solution

(a) Ca²⁺ + OH⁻ → Ca(OH)₂

(b) Al³⁺ + SO₄²⁻ → Al₂(SO₄)₃

(c) NH₄⁺ + NO₃⁻ → NH₄NO₃

3. Name these compounds: (a) FeO, (b) Fe₂O₃, (c) CuO, (d) Cu₂O

Show Solution

(a) FeO → Iron(II) oxide (Fe²⁺)

(b) Fe₂O₃ → Iron(III) oxide (Fe³⁺)

(c) CuO → Copper(II) oxide (Cu²⁺)

(d) Cu₂O → Copper(I) oxide (Cu⁺)

4. What is wrong with these formulas? (a) CaOH₂, (b) Na₂Cl

Show Solution

(a) CaOH₂ is wrong - should be Ca(OH)₂ (brackets needed for hydroxide)

(b) Na₂Cl is wrong - should be NaCl (Na⁺ + Cl⁻ = 1:1 ratio)

Key Takeaways for WAEC/JAMB/NECO

  • • Know symbols of common elements (especially Latin names: Na, K, Fe, Cu, Ag, Au, Pb)
  • • Memorize valencies: Group 1 = 1, Group 2 = 2, Group 3 = 3, etc.
  • • "Swap and drop" method: cross valencies, simplify if needed
  • • Use brackets for polyatomic ions when more than one is needed
  • • Know common polyatomic ions: OH⁻, NO₃⁻, CO₃²⁻, SO₄²⁻, NH₄⁺
  • • Variable valency elements: Fe (2,3), Cu (1,2), Pb (2,4)
