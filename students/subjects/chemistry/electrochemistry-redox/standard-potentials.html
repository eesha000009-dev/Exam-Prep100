Standard Electrode Potentials

Understanding E° values, the electrochemical series, and predicting reaction spontaneity.

Learning Objectives

  • • Define standard electrode potential (E°)
  • • Read and interpret the electrochemical series
  • • Calculate cell potential using standard potentials
  • • Predict whether a redox reaction will occur spontaneously

1 What is Standard Electrode Potential?

Standard electrode potential (E°) is the voltage measured for a half-cell under standard conditions compared to the standard hydrogen electrode (SHE).

Standard Conditions:

  • • Temperature: 25°C (298 K)
  • • Concentration: 1.0 mol/dm³ for solutions
  • • Pressure: 1 atm for gases

The Standard Hydrogen Electrode (SHE)

The SHE is assigned an E° value of 0.00 V. All other electrode potentials are measured relative to this reference. It consists of hydrogen gas at 1 atm bubbling over a platinum electrode in 1M H⁺ solution.

2 The Electrochemical Series

The electrochemical series is a list of half-reactions arranged in order of their standard electrode potentials. It helps us predict the direction of redox reactions.

More Negative E° Values

  • • Stronger reducing agents
  • • Lose electrons more easily
  • • Metals like Li, K, Na are very reactive
  • • Displace metals below them in the series

More Positive E° Values

  • • Stronger oxidizing agents
  • • Gain electrons more easily
  • • Halogens like F₂, Cl₂ are very reactive
  • • Oxidize metals above them in the series

3 Calculating Cell Potential

The Formula:

E°cell = E°cathode - E°anode

Or: E°cell = E°reduction - E°oxidation

Important Rules

  • • DO NOT multiply E° values even if you balance the equation
  • • The species with more positive E° will be reduced (cathode)
  • • The species with more negative E° will be oxidized (anode)
  • • If E°cell > 0, the reaction is spontaneous

Step 1: Write the half-reactions and E° values

Zn²⁺ + 2e⁻ → Zn     E° = -0.76 V

Cu²⁺ + 2e⁻ → Cu     E° = +0.34 V

Step 2: Identify oxidation and reduction

Zn has more negative E°, so it will be oxidized (anode)

Cu²⁺ has more positive E°, so it will be reduced (cathode)

Step 3: Calculate E°cell

E°cell = E°cathode - E°anode

E°cell = (+0.34) - (-0.76)

E°cell = +1.10 V

Answer: YES! E°cell = +1.10 V (positive, so spontaneous)

Zinc will displace copper from copper(II) sulfate solution.

4 Predicting if Reactions Will Occur

The Simple Rule

A spontaneous redox reaction will occur when:

The species being reduced has a MORE POSITIVE E°

than the species being oxidized

Reaction WILL Occur

E°cell > 0 (positive)

The reducing agent is ABOVE the oxidizing agent in the electrochemical series.

Example: Zn + Cu²⁺ → Zn²⁺ + Cu ✓

Reaction WON't Occur

E°cell < 0 (negative)

The reducing agent is BELOW the oxidizing agent in the electrochemical series.

Example: Cu + Zn²⁺ → Cu²⁺ + Zn ✗

Practice Problems

1. Will silver react with copper(II) sulfate solution?

Show Solution

Ag⁺ + e⁻ → Ag    E° = +0.80 V

Cu²⁺ + 2e⁻ → Cu    E° = +0.34 V

For Ag to displace Cu: Ag → Ag⁺ + e⁻ (oxidation)

E°cell = (+0.34) - (+0.80) = -0.46 V

Since E°cell is negative, NO reaction will occur.

2. Calculate the E°cell for: Mg + 2Ag⁺ → Mg²⁺ + 2Ag

Show Solution

Mg²⁺ + 2e⁻ → Mg    E° = -2.37 V

Ag⁺ + e⁻ → Ag    E° = +0.80 V

Mg is oxidized (anode), Ag⁺ is reduced (cathode)

E°cell = (+0.80) - (-2.37) = +3.17 V

Very spontaneous reaction!

3. Which is the stronger oxidizing agent: Cl₂ or Br₂?

Show Solution

Cl₂ + 2e⁻ → 2Cl⁻    E° = +1.36 V

Br₂ + 2e⁻ → 2Br⁻    E° = +1.07 V

Cl₂ has the more positive E° value, so it is the stronger oxidizing agent.

Cl₂ will oxidize Br⁻ to Br₂, but Br₂ cannot oxidize Cl⁻.

Key Takeaways

  • • E° values are measured relative to the Standard Hydrogen Electrode (0.00 V)
  • • More negative E° = stronger reducing agent (easily oxidized)
  • • More positive E° = stronger oxidizing agent (easily reduced)
  • • E°cell = E°cathode - E°anode (NEVER multiply E° values)
  • • If E°cell > 0, the reaction is spontaneous
