What is Factorization?

Factorization is breaking down an algebraic expression into its simplest form by finding common factors and patterns. It's the reverse of expansion.

x² + 5x + 6
=>
(x+2)(x+3)

Factorization Methods

What is GCF (Greatest Common Factor)?

It's the biggest piece that all the terms share — a number and/or variable you can take outside the parentheses.

Step-by-step: Factor 3x² + 6x

  1. Look at each term: 3x² and 6x
  2. Find what they have in common (numbers): List the factors of each number to see the largest shared one.
    • 3 → 1, 3
    • 6 → 1, 2, 3
    • Common factors: 1, 3 → largest is 3
  3. Find what they have in common (variables): Compare powers of x in each term.
    • x² has x·x
    • x has x
    • Both have at least one x, so take the lowest power: x
  4. Combine the common parts: number 3 and variable x → GCF = 3x
  5. Divide each term by the GCF to find what's left:
    • 3x² ÷ 3x = x
    • 6x ÷ 3x = 2
  6. Write the answer: 3x(x + 2)

Quick check: Multiply back: 3x·x + 3x·2 = 3x² + 6x ✓

The Magic Pattern: a² - b² = (a+b)(a-b)

This works when you have two PERFECT SQUARES being subtracted.

Step-by-step: Factor x² - 9

  1. Check if both are perfect squares:
    • x² = x·x ✓ (perfect square)
    • 9 = 3·3 ✓ (perfect square)
  2. Check if it's subtraction: x² - 9 (yes, minus sign) ✓
  3. Identify a and b: a = x, b = 3
  4. Use the pattern: (a+b)(a-b) = (x+3)(x-3)

Check: (x+3)(x-3) = x² - 3x + 3x - 9 = x² - 9 ✓

Trinomial = Expression with 3 terms

Pattern: x² + bx + c = (x + ?)(x + ?)

You need to find the two numbers that go in the blanks.

Step-by-step: Factor x² + 5x + 6

  1. Identify b and c: b = 5, c = 6
  2. Find two numbers that MULTIPLY to give c (6):
    • 1 × 6 = 6
    • 2 × 3 = 6
  3. Check which pair ADDS to b (5):
    • 1 + 6 = 7 ✗
    • 2 + 3 = 5 ✓
  4. Use these numbers: (x + 2)(x + 3)

Check: (x+2)(x+3) = x² + 3x + 2x + 6 = x² + 5x + 6 ✓

Grouping = Split into pairs and find common factors

This works with 4 terms that have a hidden pattern.

Step-by-step: Factor xy + 2y + 3x + 6

  1. Split into two pairs:
    • (xy + 2y) + (3x + 6)
  2. Factor out the common factor from each pair:
    • First pair: xy + 2y = y(x + 2)
    • Second pair: 3x + 6 = 3(x + 2)
  3. Now you have: y(x + 2) + 3(x + 2)
  4. Notice (x + 2) appears in both! Factor it out:
  5. Answer: (x + 2)(y + 3)

Check: (x+2)(y+3) = xy + 3x + 2y + 6 ✓

Worked Examples (step-by-step)

Example 1: 2x² - 8x
  1. Look at each term: 2x² and -8x
  2. Find what they have in common (numbers): 2 and 8 → common factors 1,2 → take 2
  3. Find what they have in common (variables): x² and x → lowest power is x
  4. GCF: 2x
  5. Divide each term by 2x: 2x² ÷ 2x = x, −8x ÷ 2x = −4
  6. Answer: 2x(x − 4)
Check: 2x·x + 2x·(−4) = 2x² − 8x ✓
Example 2: 4a² − 25b²
  1. Recognise the pattern: Both terms are perfect squares (4a² = (2a)², 25b² = (5b)²)
  2. Structure: a² − b² = (a + b)(a − b)
  3. Set a and b: a = 2a, b = 5b (treating whole parts as 'a' and 'b')
  4. Answer: (2a + 5b)(2a − 5b)
Check: Multiply to confirm the cross terms cancel and you get 4a² − 25b² ✓
Example 3: m² + 7m + 12
  1. Identify b and c: b = 7, c = 12
  2. List factor pairs of 12: (1,12), (2,6), (3,4)
  3. Find pair that adds to 7: 3 + 4 = 7 → choose (3,4)
  4. Answer: (m + 3)(m + 4)
Check: (m+3)(m+4) = m² + 4m + 3m + 12 = m² + 7m + 12 ✓
Example 4: ab + 3a + b + 3
  1. Group terms: (ab + 3a) + (b + 3)
  2. Factor each group: a(b + 3) + 1(b + 3)
  3. Factor out the common binomial: (b + 3)(a + 1)
  4. Answer: (a + 1)(b + 3)
Check: (a+1)(b+3) = ab + 3a + b + 3 ✓
