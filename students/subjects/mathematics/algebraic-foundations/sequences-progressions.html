What are Sequences & Progressions?

A sequence is a list of numbers in a specific order. A progression is a sequence that follows a rule. The two main types are Arithmetic Progression (AP) and Geometric Progression (GP).

Tip: AP adds the same number each time. GP multiplies by the same number each time.

Key Methods

The nth term of an AP: a_n = a + (n-1)d

a = first term, d = common difference, n = term number.

Tip: Use this to find any term in the sequence.

Sum of first n terms: S_n = n/2 [2a + (n-1)d]

a = first term, d = common difference, n = number of terms.

Tip: Use this to find the total of several terms.

The nth term of a GP: a_n = a × rn-1

a = first term, r = common ratio, n = term number.

Tip: Use this to find any term in a geometric sequence.

Sum of first n terms: S_n = a(1 - r^n)/(1 - r) (for r ≠ 1)

a = first term, r = common ratio, n = number of terms.

Tip: Use this to find the total of several terms in a geometric sequence.

Worked Examples (step-by-step)

Example 1: Find the 5th term of AP (a=3, d=2)
  1. Use formula: a_n = a + (n-1)d
  2. Plug in: a=3, d=2, n=5 → a_n = 3 + (5-1)×2 = 3 + 8 = 11
  3. Final answer: 11
Example 2: Find the 5th term of GP (a=2, r=3)
  1. Use formula: a_n = a × rn-1
  2. Plug in: a=2, r=3, n=5 → a_n = 2 × 34 = 2 × 81 = 162
  3. Final answer: 162
Example 3: Sum of first 4 terms of AP (a=1, d=2)
  1. Use formula: S_n = n/2 [2a + (n-1)d]
  2. Plug in: a=1, d=2, n=4 → S_4 = 4/2 [2×1 + 3×2] = 2[2+6]=2×8=16
  3. Final answer: 16
Example 4: Sum of first 3 terms of GP (a=2, r=2)
  1. Use formula: S_n = a(1 - r^n)/(1 - r)
  2. Plug in: a=2, r=2, n=3 → S_3 = 2(1-2^3)/(1-2) = 2(1-8)/(-1) = 2(-7)/(-1) = 14
  3. Final answer: 14
