What is Algebra?

Algebra uses letters (variables) to stand for numbers. We learn to work with expressions and simplify them by combining like terms and using rules such as the distributive property.

Key Ideas

Variables are letters that stand for numbers (e.g., x, y).

Like terms have the same variable(s) to the same power. Example: 3x and -2x are like terms; 3x and 3x² are not.

Step: When simplifying, add/subtract only like terms.

The coefficient is the number in front of a variable (3x → 3). A constant has no variable (5 is a constant).

To combine like terms, add/subtract their coefficients and keep the variable part the same.

a(b + c) = ab + ac. Multiply the term outside the bracket with each term inside.

Example: 3(x + 4) → 3x + 12.

Tip: Use distributive property to remove brackets before combining like terms.

Worked Examples (step-by-step)

Example 1: Simplify 3x + 5x
  1. Recognise like terms: 3x and 5x both have x
  2. Add coefficients: 3 + 5 = 8
  3. Answer: 8x
Example 2: Simplify 4a - 2a + 7
  1. Combine like terms (4a - 2a = 2a)
  2. Constants: 7 stays as is
  3. Answer: 2a + 7
Example 3: Expand 2(x + 3)
  1. Distribute: 2·x + 2·3
  2. Simplify: 2x + 6
  3. Answer: 2x + 6
Example 4: Simplify 5 + 2x + 3 + x
  1. Group like terms: (2x + x) and (5 + 3)
  2. 2x + x = 3x; 5 + 3 = 8
  3. Answer: 3x + 8
