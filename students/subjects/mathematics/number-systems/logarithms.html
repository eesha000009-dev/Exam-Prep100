Logarithms

Definition: log_b(a) = c ↔ b^c = a. Laws of logarithms and solving exponential equations using logs.

Worked example

Solve 2^x = 16 → x = log_2(16) = 4.

Practice

  1. Evaluate log_10(1000).
  2. Solve 10^{x} = 250 (give answer in logs).

Brief solutions

  1. log_10(1000) = 3.
  2. x = log_{10}(250).
