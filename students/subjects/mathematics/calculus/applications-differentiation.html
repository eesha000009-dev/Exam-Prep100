Maxima & Minima

Find stationary points by solving f'(x)=0. Use second derivative or sign change to classify.

Visual — turning point

A parabola with a maximum at x=2 (example f(x)=-x^2+4x).

Worked examples

1) f(x)= -x^2 + 4x → f'(x) = -2x + 4. Solve -2x + 4 = 0 → x=2 (maximum). Second derivative f''(x) = -2 < 0 confirms maximum.
2) Optimization: Given perimeter 20, rectangle sides x and (10−x). Area A(x)=x(10−x)=10x−x^2. A'(x)=10−2x → set 0 → x=5. Max area 25 (square).

Practice

1) Find stationary points of y = x^3 - 3x^2
2) If s(t) position, what is s'(t)?
3) For A(x)=10x−x^2, at x=5 what is A''(x)? Use to classify extremum.
4) For f(x) = x^3−3x^2, classify x=0: min, max or inflection?
