Key Concepts

1. DNA Basics

  • • DNA structure:
    • - Double helix shape
    • - Base pairs (A-T, G-C)
    • - Sugar-phosphate backbone
  • • DNA function:
    • - Genetic information storage
    • - Passing traits to offspring
  • • DNA location:
    • - Cell nucleus
    • - Chromosomes

2. Genes and Alleles

  • • What are genes?
    • - Units of inheritance
    • - Instructions for traits
    • - Sections of DNA
  • • Alleles:
    • - Different versions of genes
    • - Trait variations
  • • Gene expression:
    • - Making proteins
    • - Controlling traits

3. Mutations

  • • What are mutations?
    • - Changes in DNA
    • - Random occurrences
  • • Types of mutations:
    • - Substitutions
    • - Insertions
    • - Deletions
  • • Effects of mutations:
    • - Harmful changes
    • - Neutral changes
    • - Beneficial changes

4. Genetic Code Basics

  • • DNA language:
    • - Four-letter alphabet (A,T,G,C)
    • - Gene instructions
  • • From genes to traits:
    • - Making proteins
    • - Building bodies
  • • Universal code:
    • - Same in all living things
    • - Evolution evidence
