Simple Machines

Understanding devices that multiply force or change its direction

Basic Principles

Key Concepts

  • • Mechanical Advantage (MA)
  • • Effort Force vs. Load Force
  • • Work Input vs. Work Output
  • • Efficiency and Energy Loss

Important Formulas

  • • MA = Load Force / Effort Force
  • • Ideal MA = Distance Effort / Distance Load
  • • Efficiency = (Work Out / Work In) × 100%
  • • Work = Force × Distance

Types of Simple Machines

Levers

First Class

  • • Fulcrum between effort and load
  • • Example: Seesaw, scissors

Second Class

  • • Load between fulcrum and effort
  • • Example: Wheelbarrow

Third Class

  • • Effort between fulcrum and load
  • • Example: Tweezers, fishing rod

Wheels and Axles

  • • MA = Wheel radius / Axle radius
  • • Reduces friction in motion
  • • Examples:
    • - Door knob
    • - Steering wheel
    • - Water wheel
    • - Bicycle wheel

Pulleys

  • • MA = Number of rope sections
  • • Types:
    • - Fixed (MA = 1)
    • - Movable (MA = 2)
    • - Compound
    • - Block and tackle

Inclined Plane

  • • MA = Length / Height
  • • Reduces effort force
  • • Examples:
    • - Ramp
    • - Spiral road
    • - Hiking trail

Wedge

  • • MA = Length / Width
  • • Converts force direction
  • • Examples:
    • - Axe
    • - Nail
    • - Door stop

Screw

  • • MA = 2πr / pitch
  • • Converts rotation to linear motion
  • • Examples:
    • - Wood screw
    • - Bolt
    • - Car jack

Compound Machines

Common Combinations

  • • Scissors (lever + wedge)
  • • Bicycle (wheel/axle + pulley)
  • • Can opener (wheel/axle + wedge)
  • • Crane (pulley + wheel/axle)

Properties

  • • Total MA = Product of individual MAs
  • • Increased complexity
  • • Higher efficiency loss
  • • Greater versatility

Problem-Solving Examples

Example 1: Lever System

Calculate the effort force needed to lift a 500N load using a first-class lever where the effort arm is 2m and load arm is 0.5m.

Solution approach:

  1. Use MA = Effort distance / Load distance
  2. Apply principle of moments
  3. Consider efficiency losses

Example 2: Pulley System

Find the effort force needed to lift 200N using a block and tackle with 4 rope sections.

Key points:

  • MA = number of rope sections
  • Consider friction losses
  • Calculate actual effort required
