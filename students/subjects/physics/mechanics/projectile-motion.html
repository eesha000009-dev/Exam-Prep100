Projectile Motion

Understanding motion in two dimensions under the influence of gravity

Key Concepts

A projectile is any object thrown into space upon which the only force acting is gravity. Key points:

  • • Motion can be analyzed in two independent directions (x and y)
  • • Horizontal motion has constant velocity (no acceleration)
  • • Vertical motion has constant acceleration due to gravity
  • • Path followed is a parabola (in absence of air resistance)

Motion Analysis

Horizontal Motion (x-direction)

Key Equations

  • • vₓ = v₀cosθ (constant)
  • • x = (v₀cosθ)t
  • • ax = 0

Characteristics

  • • No acceleration
  • • Constant velocity
  • • Uniform motion

Vertical Motion (y-direction)

Key Equations

  • • vy = v₀sinθ - gt
  • • y = (v₀sinθ)t - ½gt²
  • • ay = -g

Characteristics

  • • Constant acceleration (g)
  • • Changing velocity
  • • Parabolic path

Important Parameters

Maximum Height

  • • hmax = (v₀sinθ)²/2g
  • • Vertical velocity = 0
  • • Time to reach = v₀sinθ/g

Time of Flight

  • • T = 2v₀sinθ/g
  • • Double the time to max height
  • • Time to return to initial height

Range of Projectile

Range Formula

  • • R = (v₀²sin2θ)/g
  • • Maximum at θ = 45°
  • • Same for complementary angles

Factors Affecting Range

  • • Initial velocity
  • • Launch angle
  • • Height difference

Special Cases

Horizontal Projection

  • • θ = 0°
  • • Initial vy = 0
  • • Range = v₀√(2h/g)

Maximum Range

  • • θ = 45°
  • • R = v₀²/g
  • • hmax = v₀²/4g

Vertical Projection

  • • θ = 90°
  • • Range = 0
  • • hmax = v₀²/2g

Practice Problems

  1. Basic Projectile

    A ball is thrown with initial velocity 20 m/s at 30° to the horizontal. Calculate:

    • Maximum height reached
    • Time of flight
    • Range of the projectile

  2. Horizontal Projection

    A stone is thrown horizontally from a cliff 45m high with velocity 15 m/s. Find:

    • Time to reach ground
    • Horizontal distance traveled
    • Velocity at impact

  3. Maximum Range

    What initial velocity is needed to achieve a range of 100m when fired at the optimal angle?

    Hint: Use R = v₀²/g at θ = 45°
