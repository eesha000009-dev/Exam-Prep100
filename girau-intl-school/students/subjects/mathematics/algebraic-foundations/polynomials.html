What is a Polynomial?

A polynomial is a math expression made by adding, subtracting, or multiplying numbers and variables (like x or y), where the variables have whole number powers (exponents).

Examples:
2x + 3 (2 terms)
x² - 4x + 7 (3 terms)
5y³ + 2y - 1 (3 terms)

Tip: Each part separated by + or - is called a term. The highest power of the variable is called the degree.

Key Methods

Degree is the highest exponent of the variable. Example: x³ + 2x² + 1 has degree 3.

Types:
Monomial (1 term, e.g. 5x)
Binomial (2 terms, e.g. x + 2)
Trinomial (3 terms, e.g. x² + x + 1)
Polynomial (any number of terms)

To find the degree, look for the biggest exponent on any variable.

Add or subtract polynomials by combining like terms. Example: (2x + 3) + (x - 5) = 3x - 2.

Like terms have the same variable and exponent. Add/subtract their numbers (coefficients).

Multiply each term in one polynomial by each term in the other. Example: (x + 2)(x + 3) = x² + 5x + 6.

Tip: Use FOIL for multiplying two binomials: First, Outer, Inner, Last.

Rewrite a polynomial as a product of simpler polynomials. Example: x² - 9 = (x + 3)(x - 3).

Factoring helps you solve equations and find roots. Look for patterns like difference of squares or common factors.

Worked Examples (step-by-step)

Example 1: Add (2x + 3) + (x - 5)
  1. Write both polynomials: (2x + 3) and (x - 5)
  2. Find like terms: 2x and x (both have x)
  3. Add their numbers: 2 + 1 = 3 → 3x
  4. Add constants: 3 + (-5) = -2
  5. Final answer: 3x - 2
Example 2: Multiply (x + 2)(x + 3)
  1. Multiply each part:
    • First: x·x = x²
    • Outer: x·3 = 3x
    • Inner: 2·x = 2x
    • Last: 2·3 = 6
  2. Add all terms: x² + 3x + 2x + 6
  3. Combine like terms: 3x + 2x = 5x
  4. Final answer: x² + 5x + 6
Example 3: Find degree of 4x³ + 2x² - x + 7
  1. List all exponents: x³ (3), x² (2), x (1), constant (0)
  2. Find the biggest: 3
  3. Final answer: Degree is 3
Example 4: Factor x² - 16
  1. Check: Is each part a perfect square? x² (yes), 16 (yes, 4²)
  2. Use the pattern: a² - b² = (a + b)(a - b)
  3. Here: a = x, b = 4
  4. Final answer: (x + 4)(x - 4)
