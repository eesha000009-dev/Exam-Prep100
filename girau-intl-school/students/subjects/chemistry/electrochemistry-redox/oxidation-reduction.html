Oxidation & Reduction

Understanding oxidation-reduction reactions, oxidation numbers, and identifying oxidizing and reducing agents.

Learning Objectives

  • • Define oxidation and reduction in terms of electrons and oxygen
  • • Calculate oxidation numbers for elements in compounds
  • • Identify oxidizing and reducing agents in reactions
  • • Balance simple redox equations

1 What are Oxidation and Reduction?

Oxidation-Reduction (Redox) reactions are reactions that involve the transfer of electrons between substances. These reactions always occur together - when one substance loses electrons, another must gain them.

Remember: OIL RIG

OIL

Oxidation Is Loss (of electrons)

RIG

Reduction Is Gain (of electrons)

Definition Oxidation Reduction
In terms of electrons Loss of electrons Gain of electrons
In terms of oxygen Gain of oxygen Loss of oxygen
In terms of hydrogen Loss of hydrogen Gain of hydrogen
Oxidation number Increases (becomes more positive) Decreases (becomes more negative)

Example: Reaction of Magnesium with Oxygen

2Mg + O2 → 2MgO
  • • Magnesium: Loses electrons (oxidized) - oxidation number changes from 0 to +2
  • • Oxygen: Gains electrons (reduced) - oxidation number changes from 0 to -2

2 Oxidation Numbers

An oxidation number (or oxidation state) is a number assigned to an element in a compound that represents the number of electrons lost or gained by an atom of that element.

Rules for Assigning Oxidation Numbers:

  1. Elements in their free state have an oxidation number of 0 (e.g., O2, Fe, Cl2)
  2. For monatomic ions, oxidation number = charge (e.g., Na+ = +1, Cl- = -1)
  3. Oxygen is usually -2 (except in peroxides where it's -1)
  4. Hydrogen is usually +1 (except in metal hydrides where it's -1)
  5. The sum of oxidation numbers in a neutral compound = 0
  6. The sum of oxidation numbers in a polyatomic ion = charge of ion
  7. Group 1 metals = +1, Group 2 metals = +2
  8. Halogens are usually -1 (unless bonded to a more electronegative element)

Step 1: Let the oxidation number of S be x

Step 2: Set up the equation (sum = 0 for neutral compound)

2(+1) + x + 4(-2) = 0

H is +1, O is -2

Step 3: Solve for x

2 + x - 8 = 0

x - 6 = 0

x = +6

Answer: The oxidation number of S in H2SO4 is +6

3 Oxidizing and Reducing Agents

Oxidizing Agent

  • • Causes oxidation in another substance
  • • Gets reduced itself (gains electrons)
  • • Accepts electrons from other species
  • • Oxidation number decreases

Examples:

O2, Cl2, H2O2, KMnO4, K2Cr2O7

Reducing Agent

  • • Causes reduction in another substance
  • • Gets oxidized itself (loses electrons)
  • • Donates electrons to other species
  • • Oxidation number increases

Examples:

H2, C, CO, SO2, Metals (Zn, Fe, Mg)

Memory Tip

An oxidizing agent is the one that gets reduced. A reducing agent is the one that gets oxidized. It sounds backwards, but think: "An oxidizing agent does the oxidizing to others, so it gets reduced itself!"

Practice Problems

1. What is the oxidation number of Mn in KMnO4?

Show Solution

K = +1, O = -2

(+1) + x + 4(-2) = 0

1 + x - 8 = 0

x = +7

2. In the reaction: CuO + H2 → Cu + H2O

Identify: (a) What is oxidized? (b) What is reduced? (c) The oxidizing agent (d) The reducing agent

Show Solution

(a) H2 is oxidized (oxidation number: 0 → +1)

(b) Cu in CuO is reduced (oxidation number: +2 → 0)

(c) CuO is the oxidizing agent

(d) H2 is the reducing agent

3. Find the oxidation number of Cr in Cr2O72-

Show Solution

O = -2, charge of ion = -2

2x + 7(-2) = -2

2x - 14 = -2

2x = 12

x = +6

Key Takeaways

  • • OIL RIG: Oxidation Is Loss, Reduction Is Gain (of electrons)
  • • Redox reactions always occur together - electrons lost = electrons gained
  • • Oxidizing agents get reduced; Reducing agents get oxidized
  • • Oxidation numbers help track electron transfer
  • • The sum of oxidation numbers equals the charge of the species
