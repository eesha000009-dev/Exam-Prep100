## Image Specifications

### Recommended Formats
- **Web**: JPEG, WebP (optimized)
- **Print**: PNG, high-resolution JPEG
- **Transparent backgrounds**: PNG

### Recommended Sizes
- **Hero images**: 1920x1080px minimum
- **Card images**: 800x600px
- **Profile photos**: 400x400px
- **Logo**: 500x500px (square) or vector

### File Naming Convention
- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive: `cultural-day.jpeg`, `headboy.jpeg`
- Include date for events: `graduation-2024.jpeg`
